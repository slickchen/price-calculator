# TRE Price Calculator

卡钟计价软件 - 支持多人实时同步、自定义折扣、快照备份

## 功能

- 🧮 **多人卡钟记录** - 每人可录入多次卡钟费用
- 💰 **自定义折扣** - 预设70/75/80/85/90折 + 全价，也可自定义任意折扣
- 🍺 **额外消费** - 饮品/烟酒等额外消费原价计算，不计入卡钟
- 🔄 **多人实时同步** - 多人同时访问，数据自动同步（2秒轮询）
- 💾 **自动保存** - SQLite持久化，每次操作自动保存
- 🔁 **重置+恢复** - 重置前自动备份，误触可一键恢复
- 📁 **快照保存** - 手动保存每次对局数据，随时查看/恢复历史
- 📋 **生成账单** - 一键生成详细账单，支持复制到剪贴板
- 📱 **手机友好** - 响应式深色UI，手机浏览器直接用

## 计算规则

```
个人合计 = 卡钟总和 × 折扣 + 额外消费
总计 = 所有人个人合计之和
```

## 安装

```bash
pip install -r requirements.txt
```

## 运行

### Web版（手机/多人访问）
```bash
python app.py
```
然后手机浏览器访问：`http://<电脑IP>:5000`

### GUI版（桌面）
```bash
python gui.py
```

## 技术栈

- Python 3.8+ / Flask / SQLite / CustomTkinter
