# TRE Price Calculator

卡钟计价软件 - 支持多人实时同步、卡钟折扣计算 + 额外消费原价计算

## 功能

- 🧮 **多人卡钟记录** - 每人可录入多次卡钟费用
- 💰 **卡钟折扣计算** - 卡钟费用总和按75折计算（折扣可调）
- 🍺 **额外消费** - 饮品/烟酒等额外消费原价计算，不计入卡钟
- 🔄 **多人实时同步** - 多人同时访问，数据自动同步（2秒轮询）
- 💾 **自动保存** - SQLite持久化，每次操作自动保存
- 🔁 **重置功能** - 一键清空数据，方便下次全新计算
- 📋 **生成账单** - 一键生成详细账单，支持复制到剪贴板
- 📱 **手机友好** - 响应式深色UI，手机浏览器直接用

## 计算规则

```
个人合计 = 卡钟总和 × 折扣 + 额外消费
总计 = 所有人个人合计之和
```

例如：
- 张三：卡钟 20+30+22=72，75折=54
- 李四：卡钟 11+15+18+23+22=89，75折=66.75
- 黄五：卡钟 20+21+34=75，75折=56.25
- 总计（卡钟75折）= 54+66.75+56.25 = 177

## 安装

```bash
pip install -r requirements.txt
```

## 运行

### Web版（手机/多人访问）
```bash
python app.py
```
然后手机浏览器访问：`http://<电脑IP>:5000`

手机和电脑需在同一局域网。多人同时访问自动同步数据。

### GUI版（桌面）
```bash
python gui.py
```

## 技术栈

- Python 3.8+
- Flask (Web后端)
- SQLite (数据持久化)
- CustomTkinter (桌面GUI)
- 原生HTML/CSS/JS (前端，零依赖)
