# TRE Price Calculator

卡钟计价软件 - 支持多人卡钟75折计算 + 额外消费（饮品/烟酒）原价计算

## 功能

- 🧮 **多人卡钟记录** - 每人可录入多次卡钟费用
- 💰 **卡钟折扣计算** - 卡钟费用总和按75折计算（折扣可调）
- 🍺 **额外消费** - 饮品/烟酒等额外消费原价计算，不计入卡钟
- 💾 **保存/加载** - JSON格式保存，随时加载历史会话
- 📋 **生成账单** - 一键生成详细账单，支持复制到剪贴板

## 计算规则

```
个人合计 = 卡钟总和 × 折扣 + 额外消费
总计 = 所有人个人合计之和
```

例如：
- 张三：卡钟 20+30+22=72，75折=54
- 李四：卡钟 11+15+18+23+22=89，75折=66.75
- 黄五：卡钟 20+21+34=75，75折=56.25
- 总计（卡钟75折）= 54+66.75+56.25 = 177

## 安装

```bash
pip install -r requirements.txt
```

## 运行

### GUI版（桌面）
```bash
python gui.py
```

### Web版（手机访问）
```bash
python app.py
```
然后手机浏览器访问：`http://<电脑IP>:5000`

手机和电脑需在同一局域网。

## 使用说明

1. **新建会话** - 点击"新建"创建新的计价会话
2. **添加客户** - 左侧输入姓名，点击"添加"
3. **录入卡钟** - 选中客户后，在右侧输入金额，点击"添加卡钟"
4. **录入额外消费** - 输入名称和金额，点击"添加"
5. **保存** - 点击"保存"将会话保存到本地
6. **生成账单** - 点击"生成账单"查看详细账单

## 技术栈
