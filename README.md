# TRE Price Calculator

卡钟计价器 - 纯前端版，GitHub Pages直接部署

## 功能

- 🧮 **多人卡钟记录** - 预设7人快捷选择，支持自定义添加
- 👥 **2V2选人** - 多选/单选人员加入对战
- 💰 **自定义折扣** - 预设70/75/80/85/90折 + 全价 + 自定义任意折扣
- 🍺 **额外消费** - 原价计算，不计入卡钟
- 🔁 **重置+恢复** - 重置前自动备份，误触一键恢复
- 📁 **快照保存** - 手动保存每次对局，随时查看/恢复历史
- 💾 **本地存储** - localStorage自动保存 + JSON导出/导入备份
- 📋 **生成账单** - 详细账单一键复制
- 📱 **手机友好** - 深色UI，离线可用

## 计算规则

```
个人合计 = 卡钟总和 × 折扣 + 额外消费
总计 = 所有人个人合计之和
```

## 使用

直接打开网页即可使用，无需安装！

**数据保存**：
- 自动保存在浏览器本地（localStorage）
- 可导出JSON文件备份，换手机导入恢复
